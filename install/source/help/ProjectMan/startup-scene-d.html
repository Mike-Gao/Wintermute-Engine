Startup scene (debug mode)

Use this property to specify the initial scene the game loads at startup when run in debug mode. This option overrides the standard startup scene option. You will probably want to start the game directly in a certain scene when developing the game and that's what this option is for.

You can either browse the scene file from here, or you can right-click the scene in the project tree pane and select "Set as startup scene (debug only)".
